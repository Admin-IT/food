    发布菜谱帮助

    ”有谱“终于可以个人发布菜谱啦~小伙伴们有没有喜大普奔的冲动？先别急，今天就先跟着小编学习一下发布菜谱的详细流程吧~

    【首先进入发布菜谱页】

    登陆自己的空间后，先点击“作品”按钮，然后点击“发布菜单”，如下图所示：

    【然后就开始专心写菜谱吧】

    1.菜谱名称：先选择“原创”或“转载”，再写上与菜谱内容相符的标题，比如红烧肉、鱼香肉丝，字数在3—10字为宜。

    2.属性：根据下拉条，一个个选择就可以啦，6个位置，不要漏掉任何一个哦。

    3.成品图：为了最好的展示效果，尺寸要求为328*440像素。右侧心情可以写菜品特色、口感、心得、来历等。

    4.记录所有原材料：食材的名称和用量都要写上，比如：白菜500克、植物油2汤匙，所有食材的量词要用：克、毫升、个、茶匙、汤匙等汉字来写。

    5.步骤：要有步骤图和文字描述，每个步骤可以选择所用食材和时间。

    6.烹饪小技巧：填写你做这道菜的小技巧或小贴士，若有食材的选购技巧和营养功效也可以写进去哦~

    7.完善菜谱标签：标签通常有功效、人群、菜系、口味、节日节令、场景等主要类型，标签的正确填写，能提升这道菜的曝光度和受欢迎度！

    8.点击 “搞定，发布!” 就可以啦，然后耐心等待管理员的审核吧 O(∩_∩)O ~

    了解啦！马上就去上传菜谱！
